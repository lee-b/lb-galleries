Kintassa Galleries - A Wordpress Plugin

Introduction

This plugin provides an easy, intuitive interface for creating image galleries/slideshows. Features include:

  • Multiple galleries
  • Multiple images per gallery
  • Titles and other metadata for images
  • Gallery size specification at the gallery level -- images are automatically resized if necessary
  • Choose how to present your gallery from various built-in GalleryApplets, including:
    • Automated slideshows
    • Manual slideshows with prev/next buttons
    • Manual slideshows with image numbers.
  • GalleryApplets are HTML/PHP template-based, allowing easy modification of generated galleries by web designers
  • Plugins supported, for custom GalleryApplets to meet future or unforeseen needs

Installation

Installation is straightforward: unpack the archive to your wp-plugins folder, and make sure your webserver can write to the following subdirectories:

  • uploads
  • cache

Then, simply enable the "Kintassa Galleries" plugin within Wordpress's admin area.

Usage

Administration

The Kintassa Galleries Table

First of all, make sure the plugin is installed and activated. To do this, see the Installation section, above.

  • Log into your wordpress site's admin area
  • You should see a "Galleries" option in the menu. Click this option. An initially empty table of galleries will be displayed, with an Add Gallery button underneath.
  • Click the Add Gallery button. This will take you to the Add Gallery form.

The Add Gallery form

Simply fill in some basic details about the gallery you'd like to setup, and choose the Confirm button. Once you do this, you will be able to add images in a separate step. Add Gallery options are as follows:

Name
Simply the name of the gallery. This should be lowercase, and all one word, as it's used within the generated webpages of your site, to allow individually styling the gallery.
Width and Height
These are simply the size of the gallery, in pixels. Images uploaded to the gallery will be shown with the dimensions you use here. Note that it is fine to change these options after uploading images -- Kintassa Galleries will adjust image dimensions whenever needed.
Display method
Manual slideshow
Displays a gallery with prev and next buttons, allowing the user to manually browse through a gallery in a pre-determined order. Images are scrolled from one to the next when changing.
Pager slideshow
Displays a gallery with image numbers, allowing the user to browse in any order they prefer. Images are scrolled from one to the next when changing.
Slideshow
A fully-automated slideshow which presents new images after a short delay. Images are faded between one and the next

Editing a gallery, and finding the table of gallery images

Locate the Gallery Table again (see above). Find the gallery you want to edit in the listings. Choose edit on the row of the gallery you wish to change.

From here the options are the same as when adding a new gallery, except that you will also see a table of images within the gallery. To change gallery options such as the width and height, simply modify the settings here, and choose Confirm.

Adding images

To add a new image to a gallery, edit the gallery (choose edit in the main galleries table), then choose Add Image. Options presented when adding a new image are as follows:

Sort Priority
Provides a quick way of moving an image to a new position. To use it, check the current Sort Orders on the Edit Gallery page, and note the number needed to place your image in the correct position. Sort orders begin at 1 (the first image) and ascend to the last image, so generally, entering 0 will move the image to the top, whilst entering 1000 will move the image to the bottom. NOTE: For most purposes, it is safe to ignore this option.
Title
Name of the image
Image
Choose the image file you want to upload. It is safe to ignore this option if only editing other options when you've already uploaded an image.
Description
This is provided for flexibility, in case you would like to add long descriptions to your galleries' images and display them via GalleryApplets (plugins), however the default display methods do not currently display this field.

Sorting images

The image display order within a gallery initially defaults to being alphabetical. Sort buttons (up and down arrows) are provided in the gallery image table. Simply click the up arrow to move an image higher, and the down arrow to move an image lower within the list. The first image in the list will be displayed initially (and when no javascript is available), with the others following in sequence. Note this this can vary depending on the gallery display mode chosen, however: some plugins allow users to browse images any order.

Inserting galleries into Pages, Posts, and Templates

Kintassa galleries are very simple to insert once setup in the admin area. Just make note of the gallery ID from the main Kintassa Galleries table in the wordpress admin area. Then, in posts or pages, you can simply enter a shortcode to add the gallery using that ID number, like so:

[kintassa_gallery id="3"]

In PHP template files (when creating Wordpress themes), you can use the kintassa_gallery($id) template tag (function) instead. For example:

<?php kintassa_gallery(3) ?>

Would insert the Kintassa Gallery which has id number 3.

Your gallery will be inserted according to the settings in the admin area. To adjust the width, height, or display method after adding a gallery to a page, just change the settings in the admin area. The page will update automatically.
